# Todo-List-App-Using-NodeJs-ExpressJs

Created an Todo List App using Nodejs Express js

User can Add or Delete task using the add and delete button.

User will have to provide 3 inputs to add a task.
  1. Date
  2. Category
  3. Description
  
## Technologies Used
* HTML
* CSS
* JavaScript
* ExpressJs
* MongoDB
